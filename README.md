# 🎮 Mini Games Hub

Mini Games Hub is a browser-based gaming website that contains several fun mini-games in one place. 
The website allows users to quickly play simple games without installation. 
It is built using HTML, CSS, and JavaScript and runs directly in the browser.

## ✨ Features

- Multiple mini-games in one website
- Responsive design for desktop and mobile
- Dark/Light theme toggle
- Background music with sound control
- Random game button
- Best score saving using LocalStorage

---

## 🕹️ Games Included

### Tic Tac Toe
A classic 3×3 board game where the player competes against computer and allow to choose symbol (X or O) in the menu.

### Memory Game
Players flip cards and try to match pairs in the shortest time. Player can choose its category such as fruit or geometry in the menu.

### Sliding Puzzle
A tile puzzle where players arrange tiles in the correct order. Player can choose grid size such as 3x3, 4x4 and 5x5 in the menu.

### Kuku Kube
A color perception game where players must find the slightly different colored square. Player can choose difficulty level in the menu.
