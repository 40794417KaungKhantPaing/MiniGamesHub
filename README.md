# 🎮 Mini Games Hub

Mini Games Hub is a browser-based gaming website that contains several fun mini-games in one place. 
The website allows users to quickly play simple games without installation. 
It is built using HTML, CSS, and JavaScript and runs directly in the browser.

---

## ✨ Features

- Multiple mini-games in one website
- Responsive design for desktop and mobile
- Dark/Light theme toggle
- Background music with sound control
- Random game button
- Best score saving using LocalStorage

---

## 🕹️ Games Included

### Tic Tac Toe
A classic 3×3 board game where the player competes against computer.

### Memory Game
Players flip cards and try to match pairs in the shortest time.

### Sliding Puzzle
A 4×4 tile puzzle where players arrange tiles in the correct order.

### Kuku Kube
A color perception game where players must find the slightly different colored square.

---

## 🧰 Technologies Used

- HTML5
- CSS3
- JavaScript
- Font Awesome
- LocalStorage API

---

## 🚀 How to Run the Project

1. Download or clone the repository.
2. Open the project folder.
3. Open **index.html** in a web browser.

---

# 🌐 Deployment

The website can be deployed using **GitHub Pages**.

Steps:
1. Upload the project to a GitHub repository.
2. Open **Repository Settings**.
3. Enable **GitHub Pages**.
4. Select the main branch as the deployment source.

The website will then be available online.
